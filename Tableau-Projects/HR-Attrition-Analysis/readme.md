# HR Attrition Analysis Dashboard (Tableau)

##  Project Overview

This Tableau dashboard analyzes **50,000 employee records** to understand workforce trends, employee attrition, salary patterns, department performance, and promotion impact.

The dashboard is fully interactive and helps HR teams make data-driven decisions.

##  Business Objective

This dashboard helps answer key HR questions:

- Which department has highest attrition?
- How many employees are active vs left?
- Does promotion delay increase attrition?
- Which employees have highest income?
- Gender workforce distribution?
- Work-life balance across job roles?
- Department-wise average experience?

##  Tool Used

- **Tableau Desktop**
- Data Source: HR Dataset (50,000 records)
