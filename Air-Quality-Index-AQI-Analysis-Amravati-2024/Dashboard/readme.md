# Power BI Dashboard

This folder contains the Power BI dashboard file for the AQI Analysis project.

## File Included

## Dashboard Features
- KPI Cards

- Quarterly AQI Analysis
- AQI Category Distribution

- Interactive Filters

Open the .pbix file in Power BI Desktop.
